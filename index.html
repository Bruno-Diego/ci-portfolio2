Welcome to my Snake Game! In this game you control a Snake (green square) searching for apples (red circle) using the arrow keys in your keyboard!

Start the game pressing any direction and try to eat as many apples as you can without hitting the dashed border of the game area or your own body because if you do, the game is OVER!
